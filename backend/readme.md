## 配置

首先得对database里面的信息进行配置

设置正确的user，password。

sa用户名可能处于未开启的状态，需要自己去sql server management studio里面开，具体细节问ai

设置好了之后单独运行database可以测试是否连接成功

------------------------------------------------------------------------

## 写人员接口

在routers里面写自己对应人员的接口

可以参考energymanager代码

------------------------------------------------------------------------

## 测试

写好自己的人员接口后，在bash里面输入

uvicorn backend.main:app --reload
